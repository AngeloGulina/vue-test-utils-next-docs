# vue-test-utils-next

The next iteration of Vue Test Utils. Based of the work in vue-testing-framework.

# Development

It's a pretty small codebase at the moment. Get started by running yarn install. You can run the tests with yarn test. That's it!

Lots to do. Most of the ideas and roadmap is in Notion. See issues here for some basic TODOs.

# Working with .vue files

There is vue-jest for loading .vue files into Jest for Vue 2. I'm not sure how much work it would be to migrate it to use @vue/sfc-compiler. To get going quickly, I hacked together a simple Jest transformer using @vue/sfc-compiler, find it here. It's probably bad, but it's enough to let us use .vue files in Jest in this project. I am finding render functions more ideal, since you can inline them in the tests and get IDE completion, but both are fine.
